# newcj_rime

自由大新倉頡RIME輸入方案

將檔案COPY到c:\Program Files (x86)\Rime\weasel-0.13.0\data\目錄下，再執行重新部署，即可在選單內找到該輸入法

自由大新 RIME輸入方案 : newcj.schema.yaml , newcj.dict.yaml 
自由大新+傳統倉頡 RIME輸入方案 : newcj+cj.schema.yaml , newcj+cj.dict.yaml
